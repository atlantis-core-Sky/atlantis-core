# Contributing to Atlantis-NEXUS

Hey, thanks for even reading this! Whether you're reporting a bug, suggesting a feature, or writing code, you're awesome.

## Bugs & Issues

- Check existing issues first – someone might already be on it.
- When you open a new one, please include:
  - What happened
  - What you expected
  - Steps to reproduce (logs are gold)

## Feature Ideas

- Open an issue with the label `enhancement`.
- Tell me what problem you're solving. I'm all ears.

## Code Contributions

1. Fork the repo.
2. Create a branch: `git checkout -b your-feature`
3. Write your code. Keep it clean.
4. Run `cargo build` to make sure it compiles.
5. Commit with a short but clear message.
6. Push and open a Pull Request.

I'll review as soon as I can. Let's make this project better together.

## Style

- Rust: `cargo fmt` before committing.
- Python: PEP 8 (I'm not super strict, but try to keep it readable).
- TypeScript: Prettier will handle it.
